Подробное описание

template<class T>
class IncludedExpansion< T >

IncludedExpansion класс добавляющий к заданному классу поле с именем, и расширяющий метод print().

Исходный класс включается как параметр нового. Требования к исходному классу:

  • класс должен иметь конструктор принимающий два числа int.
  • класс должен иметь метод print().
